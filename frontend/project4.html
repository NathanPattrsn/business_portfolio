Bike Size Finder Web Application

This web application was created to help users find the correct bike size based on their height and inseam measurements. It features a user-friendly interface where users can input their measurements, and it provides instant feedback on the recommended bike size. The design is clean and modern, ensuring ease of use across all devices. It is still in early development.

Tools & Technologies Used

  • Vue.js 3 – Built using Vue 3 with the Composition API and single-file components for modular development.
  • Vue CLI – Used for project scaffolding, local development, and optimized production builds.
  • JavaScript (ES6+) – Core application logic including state handling and dynamic rendering of components.
  • ESLint & Babel – Ensured code quality, linting, and modern JavaScript transpilation during development.
  • Responsive Design – Layout and styling tailored for both desktop and mobile using modern CSS techniques.
  • Git & GitHub – Version control and collaboration via GitHub for code management and hosting.

Key Features

  • Search for professional bike fitters by region or name.
  • Displays detailed cards for each expert with contact info and location.
  • Client-side rendering with fast UI interaction and filtering.
  • Fully responsive layout for both mobile and desktop users.

What I Learned

  • How to structure a project using the Vue 3 Composition API and single-file components.
  • Working with reactive data and conditional rendering in Vue.
  • Building responsive interfaces without a CSS framework.
  • Implementing dynamic search/filter functionality in a clean, performant way.
  • Using Git for version control and GitHub for project deployment or showcasing code.
