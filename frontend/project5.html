Rainbow Six Siege Operator Showcase — Interactive Web App

A responsive and immersive web application themed around Ubisoft’s Rainbow Six Siege, showcasing in-game operators, their unique abilities, roles, weapons, and interactive 3D weapon previews. Designed to mimic the game’s style and enhance user experience with smooth navigation and modern UI techniques. It is still in early development.

Tools & Technologies Used

  • React.js – Built using the React framework with functional components and React Router for dynamic page navigation.
  • Tailwind CSS – For utility-first, responsive, game-themed styling with custom color schemes and blur effects.
  • JavaScript (ES6+) – Core logic and interactivity including state management and rendering.
  • React Router – Enables dynamic routes for operator detail pages.
  • Sketchfab Embed API – Integrated 3D weapon models using Sketchfab’s iframe embeds, allowing users to rotate, zoom, and interact with each weapon visually.
  • Netlify – Deployed the fully static site with drag-and-drop or GitHub CI/CD pipeline.
  • Git & GitHub – Version control and collaboration readiness.
  • Responsive Design – Fully responsive layout using CSS grid and flexbox, optimized for both desktop and mobile.

Key Features

  • Operator Detail Pages – Each operator includes a portrait, detailed description, role, speed, armor, and gadgets.
  • Grid-Based Gallery – Displays all operators in a clean, multi-column grid layout (8 per row where possible).
  • Dynamic Routing – Allows users to click into any operator card and view full details on a separate route.
  • 3D Weapon Viewer – Interactive weapon previews powered by Sketchfab embeds, with seamless integration into the UI.
  • Blurred Background Info Cards – Mimicking the game's aesthetic, operator cards feature clean, blurred overlays for ability and name.
  • Netlify Hosting – Fast-loading and optimized deployment with support for single-page app routing.
